# RepoMem Social Preview Layout

## Canvas

- Size: `1280 x 640`
- Safe area: keep all text within the center `1080 x 440`
- Visual direction: clean, technical, high-contrast, not playful

## Primary Copy

- Main title: `RepoMem`
- Subtitle: `Make repositories remember`
- Supporting line: `A persistent memory layer for code repositories and coding agents`

## Chinese Variant

- Main title: `RepoMem`
- Subtitle: `让仓库真正拥有记忆`
- Supporting line: `一个面向代码仓库与 AI Coding Agent 的持久化记忆层`

## Recommended Layout

### Left Block

- Put `RepoMem` large and bold in the upper-left quadrant
- Put the subtitle directly below it
- Put the supporting line under the subtitle in smaller text

### Right Block

Use a minimal structural motif instead of a screenshot:

- `temp`
- `merge`
- `persist`

Show them as three stacked or connected blocks to imply the memory promotion model.

## Suggested Hierarchy

1. `RepoMem`
2. `Make repositories remember`
3. `temp -> merge -> persist`
4. `A persistent memory layer for code repositories and coding agents`

## Visual Style

- Background: dark graphite or deep green-black gradient
- Accent colors:
  - deep teal
  - muted violet
  - desaturated green
- Typography:
  - geometric sans for title
  - neutral sans for support text
- Mood:
  - infrastructure
  - calm confidence
  - technical credibility

## Do Not Do

- do not use cartoon agent icons
- do not show browser screenshots
- do not overload with badges or logos
- do not add too many claims
- do not make it look like a generic AI SaaS banner

## One-Line Goal

The image should make people understand this in under two seconds:

`RepoMem gives AI-driven repositories durable memory.`
